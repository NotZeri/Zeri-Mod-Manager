# Zeri Mod Manager

> BepInEx installer for Unity 2022 Mono games.






## Features
- Automatic install
- Plugin support
- Lightweight
- Unity Explorer




## Installation

1. Download release
2. Run installer
3. Select game folder
4. Press install
