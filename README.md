# Zeri Mod Manager

> BepInEx installer for Unity 2022 Mono games.






## Features
- Automatic install
- Plugin support
- Lightweight
- Unity Explorer

<img width="2766" height="911" alt="preview" src="https://github.com/user-attachments/assets/cdcbcf95-9684-42ef-85fe-a75133a9f966" />


## Installation

1. Download release
2. Run installer
3. Select game folder
4. Press install
